# Bags Play CLI

> Programmable fees for Bags tokens.

Bags Play is a composable automation platform for building reusable Apps (Bags Apps) in the [Bags](https://bags.fm) Solana ecosystem. Create, share, and compose token fee automations that run on the cloud.

## Install

```bash
curl -fsSL https://play.bags.fm/install.sh | bash
```

Or install a specific version:

```bash
curl -fsSL https://play.bags.fm/install.sh | bash -s -- v0.0.23
```

### Supported Platforms

| Platform | Architecture |
| -------- | ------------ |
| macOS    | arm64, x64   |
| Linux    | arm64, x64   |

## Usage

```bash
# Authenticate with your Bags account
bags login

# Check who you're logged in as
bags whoami

# Show help
bags --help
```
